# Topic to Room Mapping  
  
## Overview  
Each room in the game is assigned to a specific learning topic.  
This allows structured progression and easy extension.  
  
---  
  
## Rooms and Topics  
  
| Room ID | Topic | Description |  
|--------------|---------------|--------------------------------------|  
| room_intro | Basics | Introduction to the game and concepts |  
| room_java | Java | Java fundamentals and syntax |  
| room_db | Database | SQL and database basics |  
| room_wirtschaft | Security | Basic Wirtschaft concepts |  
  
---  
  
## Design Decisions  
  
- Each room focuses on one main topic  
- Topics are clearly separated  
- Rooms can be extended with more subtopics  
- New rooms can be added easily  
  
---  
  
## Future Extension  
  
- Multiple teachers per topic  
- Difficulty levels (easy, medium, hard)  
- Dynamic room unlocking
